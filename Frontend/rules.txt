UI/UX Rules & Design System
Sistem Penyewaan Peralatan Bengkel (Astro + Tailwind)

Gunakan summary ini sebagai guideline tetap untuk AI agent lain agar desain dan coding tetap konsisten.

PROJECT OVERVIEW

Nama sistem:

Sistem Penyewaan Peralatan Bengkel Skala Kecil untuk Kebutuhan Perjalanan dan Perbaikan Darurat

Tujuan:

Memudahkan pengguna menyewa alat bengkel
Fokus mobile
Cepat digunakan saat kondisi darurat
UX sederhana dan minim langkah

Tech stack:

Astro
TailwindCSS
Vue.js
Responsive Design
Mobile First
DESIGN PRINCIPLES
1. Mobile First

Karena user kemungkinan:

sedang di jalan
menggunakan HP
kondisi darurat

Maka:

tombol besar
CTA jelas
sedikit teks
alur cepat
maksimal 2–3 klik
2. Emergency Friendly

UI harus:

mudah dipahami
tidak ramai
warna kontras
cepat dibaca

Prioritas:

Cari alat
Sewa alat
Bantuan darurat
3. Clean Modern UI

Style:

clean
rounded
soft shadow
gradient modern
spacing lega

Inspirasi:

Gojek
Tokopedia
Traveloka
Grab

TYPOGRAPHY

Font:

Inter
Poppins

Rules:

Heading: bold
Body: medium/regular
Jangan terlalu kecil

Sizing:

h1 → text-4xl / text-5xl
h2 → text-3xl
body → text-base
caption → text-sm
SPACING RULES

Gunakan:

rounded-2xl
rounded-3xl
p-4
p-6
p-8
gap-4
gap-6

Jangan:

terlalu rapat
terlalu banyak border
COMPONENT RULES
BUTTON

Primary button:

class="bg-orange-500 hover:bg-orange-600 text-white rounded-xl px-4 py-3 font-semibold transition"

Rules:

rounded-xl minimum
tinggi tombol besar
mudah ditekan di mobile
INPUT
class="w-full rounded-xl border border-gray-300 px-4 py-3 focus:ring-2 focus:ring-orange-500 outline-none"

Rules:

clean
simple
focus state jelas
CARD
class="bg-white rounded-2xl shadow-md hover:shadow-xl transition"

Rules:

gunakan shadow lembut
hover animation ringan
padding lega
PAGE STRUCTURE
LOGIN PAGE

Layout:

Desktop:
kiri → hero section

Mobile:
hanya login form

Hero:

gradient orange → red

DASHBOARD

Structure:

Navbar
Sidebar
Main Content

Isi:

search

status penyewaan
tombol emergency
TOOL CARD

Isi:

gambar alat

harga

tombol sewa

Status:

Login
→ Dashboard
→ Cari Alat

→ Pengembalian

RESPONSIVE RULES
MOBILE

Prioritas utama.

Gunakan:

single column
bottom navigation
floating CTA
full width card
DESKTOP

Gunakan:

sidebar kiri
grid tools
statistik dashboard
ANIMATION RULES

Gunakan animasi ringan saja.

Contoh:

class="transition duration-300 hover:scale-105"

Hindari:

animasi berat
terlalu banyak motion
ICON RULES

Gunakan:

Lucide Icons
Heroicons

Style:

outline
clean
minimal
ACCESSIBILITY RULES

Wajib:

contrast jelas
tombol besar
text readable
focus visible
FOLDER STRUCTURE

 ├── components/
 ├── layouts/
 ├── pages/
 ├── styles/

NAMING RULES

Gunakan:

PascalCase → component
camelCase → variable/function
kebab-case → file utilitas kecil

Contoh:

ToolCard.astro
EmergencyButton.astro
